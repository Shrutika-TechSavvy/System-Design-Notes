**A CDN**
- Content Delivery network or content Distribution network is a geographically distributed network of proxy servers and corresponding data centers ie it caches the content closer to user.

Suppose your website is hosted on a single server ie the US's us and user is requesting any service or is accessing the website from India **IN**.
Now , what will happen is request will travel through thousands of kilometres, high latency in response, slow loading and only one server gets loaded by many requests.
